# 💚 그린라이트 감지기 (Love Signal Detector)

### 👋 프로젝트 소개
썸 타는 상대방과의 카톡 대화를 분석하여 호감도를 판별해 주는 AI 서비스입니다.
OpenAI의 GPT-4o 모델을 활용하여 프롬프트 엔지니어링을 수행했습니다.

### 🛠 사용 기술
- **Language:** Python
- **AI Model:** OpenAI GPT-4o
- **Web Framework:** Streamlit (Optional)

### 🚀 실행 방법
1. API Key 발급 (OpenAI)
2. 'love_signal_analysis.ipynb' 실행 또는... `streamlit run app.py`

### 📊 프롬프트 전략
- **Role:** 연애 전문 유튜버 페르소나 부여
- **Analysis:** 대화의 길이, 질문 빈도, 이모티콘 사용 여부 분석
